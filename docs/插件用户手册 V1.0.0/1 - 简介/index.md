## **1.1 AnimoXtend是什么？**

AnimoXtend 是一款面向 3D 动画与创意制作的 Blender 插件，不仅提供高效的动画生成和重定向工具，还专注于智能化与自动化工作流。插件支持批量处理和自动优化，极大地减少繁琐操作，提升创作效率。AnimoXtend 不只是工具的集合，更是为动画师和创作者打造的创新平台，以提升创意表达和技术效率为核心，真正实现从创作到优化的全流程智能支持。
