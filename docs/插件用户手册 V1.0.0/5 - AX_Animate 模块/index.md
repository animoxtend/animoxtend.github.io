# 5 - AX_Animate 模块

> [[toc]]

AX_Animate模块用于对骨骼动画进行生成与编辑，方便动画师能够在Blender中便捷的执行：创建长序列动画、重定向动画到不同角色、混合多段动画和批量编辑动画等操作，该模块共包含4个子模块：
